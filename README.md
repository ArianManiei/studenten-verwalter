# studenten-verwalter

Einfache Studentenverwaltung in Java (Konsolen-App).
- Studenten hinzufügen (ID-Validierung: Initialen + Geburtsjahr)
- Note & Kurs hinzufügen (Exception Handling)
- Alle Studenten anzeigen
- Durchschnittsnote berechnen
- Speichern in CSV: `out/students.csv`

## Run
```bash
javac -d bin src/Projekt/*.java
java -cp bin Projekt.Main
